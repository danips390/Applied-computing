El presente repositorio cuenta con múltiples archivos. El workflow general para el procesamiento de los textos se encuentra disponible en el archivo Workflow.jpynb, para la ejecución de este código es necesaria la instalación de librerías.

Se recomienda hacer el siguiente pip install para la descarga de las librerías:
!pip install pandas numpy torch scikit-learn transformers datasets evaluate seaborn matplotlib

En los otros archivos jpynb se encuentran los que fueron utilizados para la descarga masiva de los textos, el preprocesamiento de los textos y el análisis de resultados respectivo.
